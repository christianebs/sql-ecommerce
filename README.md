<h2><img width="20px" src="https://github.com/christianebs/christianebs/assets/108686840/45469bba-2011-45d3-8d45-d03d56753e32"> Sobre o Projeto </h2>

Neste projeto, foi criado um modelo lógico de banco de dados utilizando o MySQL. O objetivo era replicar a modelagem de um banco de dados para um cenário de e-commerce e traduzir os requisitos do cenário em uma estrutura lógica coerente usando o MySQL.
